# 🏠 Sumika

**Nowoczesny system do zarządzania najmem nieruchomości.**

Sumika to projekt składający się z aplikacji webowej, mobilnej oraz REST API, którego celem jest ułatwienie zarządzania nieruchomościami, lokalami, umowami najmu, płatnościami oraz zgłoszeniami serwisowymi.

## Technologie

* ASP.NET Core Web API
* Angular
* Flutter
* PostgreSQL
* Entity Framework Core
* Docker

## Repozytoria

* **backend** – REST API
* **frontend** – aplikacja webowa
* **mobile** – aplikacja mobilna
* **docs** – dokumentacja techniczna
